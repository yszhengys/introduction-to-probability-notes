# Simulations

这里放 Python 概率模拟、小实验和 Jupyter Notebook。

建议先从小脚本开始，例如：

- 掷骰子模拟
- 伯努利试验和二项分布
- 大数定律可视化
- 条件概率模拟

## Suggested Files

- `*.py`: 简单、可复现的小模拟。
- `*.ipynb`: 需要图表、解释文字和交互输出的 notebook。
- `requirements.txt`: 项目依赖。

## Setup

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r Simulations\requirements.txt
```
